Movit Friends

I developed this with MVVM architecture as a map-based social
media application. In the application, users can view and
communicate with other registered users on the map with various
filters. I've added in-app purchases for features like premium, credit
purchase, and ad removal. I used technologies such as Firebase,
Google AdMob.

Developed with:

  • Firebase
  • Google Maps
  • MVVM Architecture
  • Jetpack Compose
  • Kotlin Coroutines
  • DataStore Preferences
  • Dagger Hilt
  • Live Data

TCP Sudoku

This is an android sudoku game that I developed with java socket
programming. In the game, you can play sudoku simultaneously with
a mobile user or a computer user, and you can message at the same
time. I used the Singleton design pattern for events such as moving
data between activities.

Developed with:

  • Java Sockets
  • Singleton Pattern
